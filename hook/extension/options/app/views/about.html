StravaPlus extension is just adding some features that riders and runners might like to see in their Strava web pages. If these features could be added by default in Strava... It could be cool! Consider getting Strava Premium. There are a ton of features that you will love. Remember, this extension is only meant to enhance the experience of all users (Free and Premium) and hopes to keep users loyal in Strava.
